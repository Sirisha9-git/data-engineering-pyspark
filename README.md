## ⭐ About
This repo is designed as a practical reference for PySpark-based data engineering workflows, interview preparation, and hands-on learning.
---
## 🔍 Topics Covered
- PySpark DataFrames and transformations  
- Joins and aggregations  
- Window functions (ranking, deduplication, latest record logic)  
- CDC patterns (inserts, updates, deletes)  
- SCD Type 2 logic (PySpark-based)  
- Performance tuning basics (partitioning, caching, broadcast joins)  
- Working with multiple file formats (CSV, Parquet, JSON, Delta, ORC)
---
## ▶️ How to Use
1. Open any folder under `notebooks/` to explore topic-specific PySpark examples.
2. Open the corresponding folder under `notes/` to read explanations for the notebook examples.
3. Use the `data/` folder for sample input files in different formats.
